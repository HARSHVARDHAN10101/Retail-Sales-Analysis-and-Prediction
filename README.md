# 🛍️ Retail Sales Data Analysis & Sales Prediction

## Project Overview

This project analyzes a real-world retail sales dataset to discover business insights and build a Machine Learning model for predicting sales.

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

---

## Project Workflow

- Data Cleaning
- Exploratory Data Analysis
- Data Visualization
- Correlation Analysis
- Random Forest Regression
- Model Evaluation

---

## Visualizations

- Sales by Category
- Profit by Category
- Sales by Region
- Profit by Region
- Top States by Sales
- Sales Distribution
- Profit Distribution
- Correlation Heatmap
- Sales vs Discount
- Profit vs Discount

---

## Machine Learning

Algorithm Used:

- Random Forest Regressor

Evaluation Metrics:

- MAE
- RMSE
- R² Score
